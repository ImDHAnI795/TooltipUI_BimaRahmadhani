# Tooltip Navigation UI

A modern and responsive Tooltip Navigation UI built using only HTML and CSS.

This project demonstrates how to create interactive tooltips that appear above navigation menu items when hovered. The tooltips are implemented entirely with CSS using positioning, transitions, and hover effects without any JavaScript.

The design follows modern UI principles and includes smooth animations, responsive layouts, and clean typography to create a professional user experience.

## Tooltip Animations

### Fade-In Effect

The tooltip gradually appears when hovering over a navigation item.

### Slide-In Effect

The tooltip smoothly slides upward into view.

### Scale-In Effect

The tooltip scales from a smaller size to its normal size.

These animation styles demonstrate different approaches to creating engaging user interactions using only CSS.
